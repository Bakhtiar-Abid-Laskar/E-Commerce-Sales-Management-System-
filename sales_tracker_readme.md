# 🚀 SalesTracker – AI-Powered Order & Dispatch Management Dashboard

SalesTracker is a modern, full-stack application designed to simplify and automate order tracking, invoice processing, and dispatch management. Built with a focus on performance and usability, it provides a clean, data-driven interface for managing the entire order lifecycle—from processing to delivery.

---

## 📄 Overview

At its core, SalesTracker integrates **AI-powered PDF parsing**, allowing users to upload invoices or shipping labels and automatically extract structured order data. This eliminates manual entry, reduces errors, and significantly speeds up workflow efficiency—especially for businesses handling bulk orders.

---

## 📊 Dashboard & Analytics

The platform features a **real-time analytics dashboard** that provides instant insights into:

- Total Revenue
- Order Volume
- Dispatch Rate

Orders are categorized into clear stages:

- Processing
- Packed
- Shipped
- Delivered

This enables users to monitor progress at a glance.

## 📦 Order Management System

A powerful table management system allows users to:

- 🔍 Search orders instantly
- 🧩 Filter and sort data
- 📌 Track status and delivery timelines
- 🚚 View courier and shipment details

Designed to handle **large datasets efficiently**.

## ⚡ Offline Support

SalesTracker uses **IndexedDB** to provide an **offline-first experience**, ensuring:

- Data persistence without internet
- Reliable local storage
- Seamless user experience
